# Online-Shopping-Behavior-Analysis-Power-BI
## 📌 Project Overview
This project is an interactive Online Shopping Behavior Analysis Dashboard developed using Power BI. It provides insights into customer purchasing behavior, sales performance, product categories, payment methods, and device usage to support data-driven business decisions.

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Power Query

## 📊 Features
- Customer Behavior Analysis
- Product & Category Performance Analysis
- Sales and Revenue Analysis
- Discount Analysis
- Payment Method Insights
- Device Usage Analysis
- Interactive Filters and Slicers

## 📈 Key Insights
- Analyzed customer purchasing patterns across different age groups and genders.
- Identified top-performing brands, products, and categories.
- Evaluated revenue trends, discounts, and order performance.
- Compared customer preferences based on payment methods and device types.
